Challenge Balidea module
------------------------

This is a module for the Balidea challenge, it includes:

路 Configuration form with a translatable "Text" field (textarea with wysiwyg)
and another "Integer" field that only allows integers.

路 Controller that the information configured in the previous form accompanied by
a fixed text for each of the fields (Informational text and Integer number).
  This controller must:
  - be accessible only by authenticated users
  - have a menu link in the main menu
  - paint the text using a template

路 The module must be prepared for English and Spanish.
路 The module must also add to the body of the site the class of the type of
content that is being displayed when it is a page of a node.
